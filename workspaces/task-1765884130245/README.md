# Unicorn Website

Eine charmante und farbenfrohe Website rund um das Thema Einhörner. Diese Seite ist darauf ausgelegt, Einhorn-Enthusiasten mit magischen Inhalten, Bildern und Geschichten zu begeistern.

## Projektübersicht

Dieses Projekt stellt eine einfache Webanwendung dar, die Informationen über Einhörner präsentiert. Es beinhaltet Funktionen zum Anzeigen von Einhornbildern, Geschichten und weiteren Inhalten in einem ansprechenden Layout.

## Setup

Um das Projekt lokal zu starten, stelle sicher, dass alle Abhängigkeiten installiert sind.

```bash
npm install
```

## Entwicklung

Für die Entwicklung und zum Testen der Anwendung steht ein Entwicklungsserver bereit. Änderungen werden automatisch übernommen.

```bash
npm run dev
```

## Produktion

Um die Website für die Produktion zu bauen, führe den Build-Befehl aus. Die gebauten Dateien befinden sich danach im `dist`-Ordner.

```bash
npm run build
```

## Projektstruktur

- `src/` - Quellcode der Anwendung  
- `public/` - Statische Dateien und Assets  
- `dist/` - Produktions-Builds
